# How-to Guides

How-to guides are task recipes for users who already understand the golden path.

Start with the tutorial first: [Golden Path Tutorial](../tutorials/golden-path.md).

Planned high-value how-tos:

- Run a feature snapshot.
- Run the ten-layer flow safely.
- Do a safe paper-trade review without live authority.
- Add a new mature-wheel adapter.
- Promote a lesson draft into a rule change with human attestation.

Until these are expanded, use the task map in [../README.md](../README.md) and
the module docs in [../modules/](../modules/).
