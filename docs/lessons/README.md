# Lessons

Use this folder for durable lessons distilled from ideas, proposals, receipts,
reviews, experiments, and repeated observations.

Lessons are not raw notes. They are promoted knowledge.

## Promotion Rule

Promote something into a lesson when it:

```text
repeats across multiple situations
changes future behavior
reduces a known risk
improves a module boundary
prevents repeated debate
clarifies a financial or AI failure mode
```

## Template

```text
# Lesson: <Title>

Date:
Status: active | superseded | archived
Source reviews:
Source ideas:
Affected modules:

## Lesson

What should future FinHarness remember?

## Why It Matters

What future mistake does this prevent?

## Evidence

What observations, receipts, reviews, or tests support it?

## Rule / Heuristic

What should we do differently next time?

## Where It Should Live

AGENTS.md | module doc | ADR | checklist | test | code
```
