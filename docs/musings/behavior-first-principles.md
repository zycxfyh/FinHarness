# Behavior From First Principles

This note answers a foundational question:

Can a behavior-centered ontology explain most human events?

Short answer:

```text
Yes, if behavior means observable action that changes, preserves, or signals a
state of reality inside a constraint system.
No, if behavior means guessing inner motives and treating those guesses as fact.
```

The useful unit is not "behavior alone".
The useful unit is:

```text
Actor + context + constraint + action + outcome + feedback
```

The stricter version of this thesis:

```text
Behavior is the most direct measurable interface between an actor and reality.
Inner states matter, but they are not directly observed.
Only behavior, traces, physiological signals, artifacts, and environmental
changes enter the shared evidence layer.
```

This does not deny psychology. It ranks evidence.

```text
Observed behavior > durable trace > reported intention > inferred motive
```

The point is not that motives are fake. The point is that motives should not be
the foundation when we are trying to explain reality-changing events.

## First Principles

An event becomes humanly meaningful when someone does something or fails to do
something under real constraints, and that action or inaction changes the state
space.

Most human events can be decomposed into:

```text
Constraint state -> Available actions -> Action/inaction -> State change -> Feedback
```

That is the smallest useful loop.

For one person:

```text
Body + skill + resources + information + social role + institution -> feasible behavior
```

For many people:

```text
Repeated behavior + coordination + incentives + institutions -> social event
```

For a market:

```text
Many actors + prices + observable orders + constraints + feedback -> aggregate outcome
```

For history:

```text
Population behavior + technology + institutions + geography + shocks -> path dependence
```

## Occam's Razor

Do not begin with identity labels.
Do not begin with unverifiable inner motives.

Begin with the smallest causal structure:

```text
What constraints are binding?
What actions are feasible?
What action is chosen?
What state changed?
What feedback can be observed?
```

Add variables only when they change the answer.

Useful variables:

```text
Capability: can the person do it?
Opportunity: does the environment permit it?
Institution: what rules and punishments exist?
Resources: what capital, time, health, and network are available?
Information: what evidence or signals are available to the actor?
Feedback: what happens after the action?
```

Remove variables that do not change incentives, constraints, options, or
feedback.

Motivation can be added later, but as an inferred variable:

```text
If two actors face similar constraints but repeatedly choose different actions,
then preference, belief, habit, identity, or hidden constraint may be inferred.
The inference remains provisional until it predicts future behavior.
```

## Why Behavior Is Powerful

Behavior is where reality-facing systems become visible.

You cannot directly observe all motives, beliefs, norms, institutions, trauma,
or incentives. But you can observe:

```text
actions
inactions
choices under constraint
prices paid
time spent
paths taken
objects produced
rules obeyed or broken
records left behind
environmental state changes
```

These are not perfect, but they are closer to shared evidence than imagined
inner explanations.

Examples:

```text
People queue for an exam -> exam access is scarce and expected payoff exceeds waiting cost.
People hold cash -> liquidity is preferred under current constraints.
People buy risky assets -> expected payoff or social payoff exceeds perceived cost.
People migrate -> staying cost exceeds moving cost for that actor.
People avoid medical care -> care-seeking cost exceeds expected benefit under perceived constraints.
```

The behavior is not merely the surface. It is the contact point between the
actor and reality.

Inner explanations may be useful, but they should be downstream of observable
patterns.

## Your Thesis, Made Precise

Your idea can be stated like this:

```text
Reality is changed by physical action, communication, allocation, coordination,
and non-action under constraints.
Mental states matter only when they become behavior, physiological signal,
communication, or durable trace.
Therefore, a rigorous social ontology should begin from observable behavior and
state change, not from imagined motives.
```

This is a strong methodological claim.

It says:

```text
Do not deny inner life.
Do not make inner life the first explanatory object.
Start from measurable interaction with reality.
```

This is why behavior is more fundamental than intention for our project:

```text
Intention without action may matter ethically or psychologically.
But action, inaction, and communication are what alter shared reality.
```

There is no scientifically verified "mind force" that directly changes external
reality without mediation through body, speech, tools, institutions, or physical
systems. So for explaining public events, the observable mediation layer matters
most.

## Behavior As Reality Interface

Use this model:

```text
Internal state -> body/speech/tool use -> environment/institution -> changed state
```

Only the middle and right side are directly public:

```text
body/speech/tool use
environmental traces
institutional records
changed state
```

So our evidence hierarchy should be:

```text
1. Direct observation of action or state change
2. Durable trace: transaction, document, artifact, log, location, price
3. Repeated behavioral pattern under comparable constraints
4. Self-report
5. Inferred motive
```

This does not make self-report useless. It means self-report is data, not proof.

## Evidence From Research

Several mature research traditions support a behavior-centered but contextual
model.

### 1. COM-B

The Behaviour Change Wheel puts COM-B at its center:

```text
Capability + Opportunity + Motivation -> Behavior
```

This is useful because it prevents naive explanations like "people just lack
willpower". A behavior can fail because the person lacks skill, lacks access, or
lacks motivation. These require different interventions.

For our stricter framework, COM-B is useful but should be read operationally:

```text
Capability -> demonstrated or plausibly available ability
Opportunity -> observable environmental permission or access
Motivation -> inferred variable, not first-order evidence
```

Reference:

- Michie, van Stralen, and West, 2011, "The behaviour change wheel", Implementation Science:
  https://pmc.ncbi.nlm.nih.gov/articles/PMC3096582/

### 2. Theory Of Planned Behavior

The Theory of Planned Behavior explains behavior through:

```text
Attitude toward behavior
Subjective norms
Perceived behavioral control
Intention
```

Meta-analytic reviews find that these variables predict meaningful variance in
intentions and behavior, but not all behavior. This is exactly the lesson we
need: behavioral models are powerful, but probabilistic.

Reference:

- Armitage and Conner, 2001, meta-analysis:
  https://pubmed.ncbi.nlm.nih.gov/11795063/

### 3. Ecological / Bioecological Development

Bronfenbrenner's ecological model argues that human development is shaped by
nested systems:

```text
Family/school/peers
Interactions between these settings
Indirect institutions
Culture and society
Historical time
```

This supports our graph approach. Behavior should not be stored as a single
tree path like age -> country -> ethnicity. It should be modeled as an actor
inside nested environments.

Reference:

- Overview of Bronfenbrenner's ecological systems theory:
  https://www.simplypsychology.org/bronfenbrenner.html

### 4. Limits Of Pure Behaviorism

Classical behaviorism showed the scientific value of observable behavior, but
its stronger forms underweighted mental states and internal models. Cognitive
science later corrected this.

Our model should learn from behaviorism without repeating its mistake:

```text
Observe behavior.
Infer hidden state cautiously.
Test the inference against new behavior.
```

Reference:

- Stanford Encyclopedia of Philosophy, Behaviorism:
  https://plato.stanford.edu/entries/behaviorism/

### 5. Biology And Heritability

Human traits are not purely social. Genetics, biology, development, and health
also shape behavior. Large twin-study meta-analysis supports substantial
heritability across many traits, while also showing that traits are complex and
not reducible to one cause.

Reference:

- Polderman et al., 2015, Nature Genetics:
  https://www.nature.com/articles/ng.3285

## What This Proves

The evidence does not prove:

```text
Behavior explains everything.
Demographics determine behavior.
One model can predict every person.
```

It supports a narrower and stronger claim:

```text
Most human events can be usefully explained by modeling actors, constraints,
resources, institutions, observable actions, state changes, and feedback loops.
```

That is enough to justify building a behavior ontology.

## What It Cannot Explain Alone

A behavior-first model needs other layers for:

```text
Natural disasters
Disease biology
Technological invention
Random shocks
Material scarcity
Complex systems cascades
Deep unconscious processes
Measurement error
Historical accidents
```

Even then, behavior often explains the human response.

Example:

```text
Earthquake: not human behavior.
Building code enforcement, evacuation, panic, aid distribution, reconstruction:
human behavior under constraints.
```

```text

```

Expanded:

```text

```

## The Explanation Test

An explanation is good only if it answers:

1. Why this behavior instead of another?
2. Why now?
3. Why by this actor?
4. What constraint or incentive made it likely?
5. What feedback will change the next behavior?
6. What observation would prove this explanation wrong?

If it cannot be falsified, it is storytelling.

## The Prediction Test

A behavior model should produce predictions like:

```text
If transaction cost drops, behavior X should increase.
If punishment certainty rises, behavior Y should decrease.
If social proof rises, adoption should accelerate.
If liquidity falls, risk-taking should fall or become more desperate.
If feedback becomes faster, learning speed should increase.
```

Then check the data.

## The Ethics Test

A human behavior ontology is powerful and dangerous.

Rules:

```text
Do not turn group-level probabilities into individual destiny.
Do not use ethnicity or nationality as causal explanation by itself.
Do not ignore institutions and resources.
Do not confuse moral judgment with causal analysis.
Do not use the model to manipulate people without consent or accountability.
```

The model should increase understanding and agency, not reduce people to labels.

## Final Position

Behavior is the best central lens because it is the measurable interface between
actors and external reality.

But the object of study is not bare behavior.
It is behavior in a system:

```text
Constraint state + capability + opportunity + institution + action + state change + feedback
```

This can explain a very large share of human events because most events are
made of people acting, not acting, coordinating, resisting, learning, adapting,
or failing under constraints.

The method is:

```text
Observe behavior.
Map constraints.
Record state change.
Compare alternatives.
Test predictions.
Update the model.
```

That is the disciplined version of a human behavior atlas.
